Detailed Description

response handler that builds an OPeNDAP DDX object

A request 'get dataddx for <def_name>;' will be handled by this response handler. Given a definition name it determines what containers are to be used to build the OPeNDAP DDX response object that includes attribute information found in a DAS object and data definitions found in a DDS object. It also knows how to transmit the DDX response object using the specified transmitter object in the transmit method.

The caller of set_response_object now owns the returned response object. The new response object is now owned by the response object.
